# CFD-Analysis-of-Li-Ion-Battery-Pack-Thermal-Management
Simulation and thermal analysis of a Li-Ion battery pack using Computational Fluid Dynamics (CFD) in ANSYS Fluent to optimize cooling performance and temperature distribution.
This project focuses on the thermal analysis and cooling optimization of Lithium-Ion battery packs using Computational Fluid Dynamics (CFD). The simulation was developed in ANSYS Workbench/Fluent to study airflow, temperature distribution, heat dissipation, and cooling efficiency in a 16-cell Li-Ion battery pack.

The study investigates:

Temperature distribution inside the battery pack
Airflow velocity and cooling performance
Thermal hotspot reduction
Enhanced wall treatment and k-epsilon turbulence modeling
Battery thermal management for improved safety and lifespan
Tools & Technologies
ANSYS Workbench
ANSYS Fluent
CFD Simulation
k-ε Turbulence Model
Thermal Analysis
Key Results
Maintained battery temperature within 300K – 323K
Achieved effective cooling using airflow velocity of 0.5 m/s
Improved thermal stability and reduced hotspot formation
Applications
Electric Vehicles (EVs)
Battery Thermal Management Systems (BTMS)
Energy Storage Systems
