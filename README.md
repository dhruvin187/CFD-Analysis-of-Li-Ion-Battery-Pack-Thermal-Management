# CFD-Analysis-of-Li-Ion-Battery-Pack-Thermal-Management

This project focuses on the thermal analysis of a Li-Ion battery pack using Computational Fluid Dynamics (CFD) in ANSYS Fluent.

## Features
- Temperature distribution analysis
- Airflow and cooling simulation
- Thermal hotspot detection
- Battery cooling optimization

## Software Used
- ANSYS Workbench
- ANSYS Fluent

## Results
- Maintained temperature between 300K–323K
- Improved cooling performance
- Reduced thermal hotspots

## Applications
- Electric Vehicles (EVs)
- Battery Thermal Management Systems
- Energy Storage Systems
